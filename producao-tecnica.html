[Dashboard] Busca no Dicionário Informal – Dez/2025

Interface interativa projetada para facilitar a coleta de dados do Dicionário Informal para fins de pesquisa linguística. O sistema permite a exploração da base de dados através de parâmetros variados, otimizando a identificação de padrões morfológicos e a exportação direta dos dados filtrados em formato de planilha.
